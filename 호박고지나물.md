# [호박고지나물] #

### 1. 재료준비 ###
1. 말린 호박고지를 물에 불린다. (30분이상)
2. 멸치, 새우가루 혹은 육수를 준비한다. (다시다 가능)
3. 소금, 간마늘, 당근, 파
4. 들기름, 들깨가루

### 2. 조리하기 ###
1. 팬에 들기름 한스푼 두르고
2. 멸치&새우가루 + 불린호박 + 소금 조금 + 마늘반스푼 넣고 중약불에 볶는다.
3. 물이 사라지면 물을 좀더 넣고 볶다가 뚜껑을 덮는다.
4. 당근 조금 넣고, 물 거의 없을 때 파를 넣는다. (옵션)
5. 마지막에 들기름이나 들깨가루를 추가해도 된다.
